## ADDED Requirements

### Requirement: Dashboard overview of current work
The system SHALL provide a terminal dashboard that summarizes current todos, recurring schedule items, recent journal entries, and pending follow-ups.

#### Scenario: Show dashboard overview
- **WHEN** the user requests the review dashboard in the terminal UI
- **THEN** the system displays a summary of current and upcoming tasks, recent journal notes, and outstanding follow-ups

### Requirement: Highlight next actions
The system SHALL highlight the next actionable items on the dashboard, including tasks due soon, overdue recurring items, and follow-up reminders.

#### Scenario: Display next actions
- **WHEN** the user views the dashboard
- **THEN** the system highlights tasks, recurring items, or follow-up reminders that require immediate attention
