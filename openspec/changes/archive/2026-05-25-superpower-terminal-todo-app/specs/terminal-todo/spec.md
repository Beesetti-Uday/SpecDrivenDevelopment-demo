## ADDED Requirements

### Requirement: Create and manage terminal todos
The system SHALL allow users to create, update, complete, and delete todo items from the terminal interface.

#### Scenario: Create a new todo
- **WHEN** the user submits a new todo command with a title and optional details
- **THEN** the system creates the todo item and displays its status and schedule

#### Scenario: Mark a todo complete
- **WHEN** the user marks an existing todo as complete
- **THEN** the system records the completion and updates the todo status
